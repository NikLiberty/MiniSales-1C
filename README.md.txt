MiniSales (1C Inventory System)

📌 Описание:

Учебный проект на 1С:Предприятие.

Реализована базовая система учета товаров (мини-склад).

---

⚙️ Функционал:

📦 Документы

* Поступление товаров (увеличивает остаток)
* Продажа товаров (уменьшает остаток)

🧮 Регистр накопления

* ОстаткиТоваров

🔄 Логика

* Формирование движений (приход/расход)
* Запись движений в регистр
* Корректный учет остатков

📊 Отчет

* Остатки товаров

---

🧠 Что было реализовано

* Полный цикл учета товаров
* Исправлена ошибка записи движений (ключевой момент 1С)

---

🎯 Цель

Демонстрация навыков разработки в 1С:

* работа с регистрами накопления
* проведение документов
* построение отчетов
